# SublimeLike-Xcode-Extensions
Adds Sublime-like text manipulation to Xcode via Xcode Extension

**Features**:
-
- Duplicate Selection(s)
- Undo Last Multi Cursor
- Convert Selection(s) to UPPERCASE
- Convert Selection(s) to lowercase
